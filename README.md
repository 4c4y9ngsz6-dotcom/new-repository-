# أفكار جديدة | New Ideas Project

## نظرة عامة | Overview
مشروع ويب حديث لهيكلة احترافية: صفحة رئيسية تفاعلية، صفحات فرعية (حول، أفكار)، تصميم متجاوب، وضع ليلي، وتنظيم مجلدات.

## الهيكل | Structure
```
.
├── index.html             # الصفحة الرئيسية
├── css/style.css          # الأنماط
├── js/script.js           # الجافاسكريبت
├── pages/                 # الصفحات الفرعية
│   ├── about.html
│   └── ideas.html
├── assets/images/         # الصور (أضف هنا)
├── TODO.md                # قائمة المهام
└── README.md              # هذا الملف
```

## الميزات | Features
- ✅ هيكل مجلدات منظم
- ✅ تصميم RTL للعربية + LTR
- ✅ Dark/Light Mode (localStorage)
- ✅ رسائل تفاعلية محفوظة
- ✅ تنقل سلس + متجاوب
- ✅ بطاقات أفكار قابلة للتوسع

## كيفية التشغيل | How to Run
1. افتح `index.html` في المتصفح:
   ```
   open index.html
   ```
2. استكشف الصفحات عبر القائمة.
3. قم بتبديل الثيم بالزر في الزاوية.
4. أضف أفكارك في `pages/ideas.html`!
